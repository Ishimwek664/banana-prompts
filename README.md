6. If Windows asks for permission, choose Run anyway

If you do not see the app start, check that the full folder stayed together after extraction.

## 🧰 What You Need

Before you run banana-prompts, make sure you have:

- Windows 10 or Windows 11
- A stable internet connection
- A Gemini API key
- A modern web browser
- Enough free space for the app files

For local use from source, you also need Node.js.

## 🔑 Set Up Your Gemini API Key

banana-prompts uses your Gemini API key to work.

1. Open the app folder
2. Find the `.env.local` file
3. Open it with Notepad
4. Add your Gemini API key next to `GEMINI_API_KEY`
5. Save the file

Use this format:

`GEMINI_API_KEY=your_api_key_here`

Keep the key private. Do not share it with others.

## 🖥️ Run the App Locally

If you want to run the app from the project files on your computer:

1. Install Node.js
2. Open the project folder
3. Open Command Prompt in that folder
4. Install the needed files with:

`npm install`

5. Add your Gemini API key in `.env.local`
6. Start the app with:

`npm run dev`

The app will open in your browser after it starts.

## 📌 Common Problems

### App does not open

- Check that you installed the release files fully
- Make sure you extracted the ZIP folder if there is one
- Try running the file again as admin

### App says the API key is not valid

- Check the key in `.env.local`
- Make sure there are no extra spaces
- Confirm that the key matches your Gemini account

### Browser page stays blank

- Wait a few seconds for the app to finish starting
- Refresh the page
- Stop the app and run `npm run dev` again

### `npm install` fails

- Check that Node.js is installed
- Close and reopen Command Prompt
- Make sure you are in the correct folder
